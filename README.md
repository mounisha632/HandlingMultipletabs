# HandlingMultipletabs
Drive Link:https://drive.google.com/file/d/1gyBeLoF0e0j6qB0ptu2LDUkStKAJ2oLn/view?usp=drive_link

Automated multiple browser tab handling using **Selenium WebDriver with Java. The script opens a new tab through UI navigation, switches between parent and child windows using window handles, validates the active page URL, and closes tabs in a controlled sequence.
